# CAD

## Установка

1. Создайте виртуальное окружение (если еще не создано):
```bash
python3 -m venv venv
```

2. Активируйте виртуальное окружение:
```bash
source venv/bin/activate  # для macOS/Linux
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python run.py
```

или

```bash
python3 run.py
```
